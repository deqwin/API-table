## API table
本工具旨在为 restful API 提供便捷的调试途径，为“调试——测试——部署”全过程提供顺畅的过渡切换，让开发更加高效。

#### 工作模式
工具共提供两种工作模式，mock数据模式和代理模式。
1、mock数据模式下，前端调试接收由工具自定义返回的mock数据以供调试，后台调试也可自定义请求数据以测试服务端程序；
2、代理模式下，工具作为代理服务器转发请求。
